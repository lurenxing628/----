## 建议修改（2项）

### 4. 备份机制简化

```python
# 原方案：定时备份（后台线程 + schedule库）
# 修改为：退出时自动备份

import atexit

def backup_on_exit():
    backup_manager.backup(suffix="auto")

atexit.register(backup_on_exit)
```

**理由**：后台定时线程增加复杂度，单机场景退出备份更实用
