### overdue 部分有效部分无效时降级标记可能误导

- ID: F-overdue-部分有效部分无效时降级标记可能误导
- 严重级别: 中
- 分类: JavaScript
- 跟踪状态: 开放
- 相关里程碑: M2
- 说明:

  extract_overdue_batch_ids_with_meta() 第65-68行：当 overdue_batches 列表中有部分无效条目（invalid_items > 0）时，会设置 degraded=True。但此时 ids 字段已经包含所有成功解析的有效 ID。而前端的甘特图在 degraded=True 时会禁用“仅超期”勾选并弹出警告，这可能让用户误以为所有超期标记都不可信，但实际上绝大多数 ID 已经被正确提取。不过，考虑到这种场景概率极低（需要 result_summary 中混入无效条目），且“宁可多警告”比“少警告”安全，当前实现不算错误。

  可选优化：当 invalid_items > 0 但有效 ID 占比超过 80% 时，降级文案改为“部分超期明细格式不正确，超期标记可能不完整但大部分可信”并保留“仅超期”功能。非阻断。
