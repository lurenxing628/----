# ADR-0009: Web 排产异步执行模型

- **状态**: 待决策（占位）
- **日期**: 2026-03-12
- **关联文档**: [排产优化_技术架构.md](../V1.2/排产优化_技术架构.md#4-5-4-web-排产异步执行模型待设计)

## 背景

当前 Web 排产为同步 POST，长耗时排产（>1 分钟）会阻塞 Flask 请求线程。需要在 Phase 3（Web UI 集成）启动前完成架构决策。

## 待决策点

- 架构选型：后台线程池 vs 任务队列 vs WebSocket 推送
- 进度反馈：轮询 vs 推送 vs 混合模式
- 超时与取消：用户能否中途取消正在执行的排产

## 约束

- 必须兼容 Win7 SP1 x64 + Python 3.8 单机部署
- 不引入外部消息队列（Redis/RabbitMQ），保持单机自包含

## 预计决策时间

Phase 3 启动前。
